This tool can analyze an individual ASUP file and display information about the replication contexts found on it.
It can also analyze two ASUP files and calculate the delta difference of the replication contexts they have in common. This is useful for knowing what a replication context has been doing between two specific dates.
In both cases the tool helps to identify bottlenecks on the replication by analyzing the time spent by each function performed within the replication context.
